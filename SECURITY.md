# Security Policy

## Supported Versions

As this is an archived project, no versions are currently supported or actively maintained. The project is preserved for historical and educational purposes.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :x: (Archived)     |

## Reporting a Vulnerability

Since this repository is archived:

1.  **Do not report vulnerabilities** expecting a fix. The code is static.
2.  **Educational Use**: You may use this repository to study security patterns or vulnerabilities in legacy PHP applications (e.g., SQL injection mitigation, session handling) for research purposes.
3.  **Contact**: For enquiries regarding the archival status, please contact the repository owner via GitHub issues if applicable.

> **Warning**: This project uses legacy encryption (MD5) and older PHP practices. It is **not recommended** for production deployment in modern environments without significant security upgrades.
