DEPLOYMENT INSTRUCTIONS FOR INFINITYFREE

1. Upload
   - Upload this entire 'Car' folder to your InfinityFree 'htdocs' directory using the File Manager or an FTP client (like FileZilla).
   - Your site URL will be:  http://your-domain.infinityfreeapp.com/Car/

2. Database Setup
   - Log in to your InfinityFree Control Panel.
   - Go to "MySQL Databases" and create a new database (e.g., 'carrental').
   - Note down the:
     - MySQL Hostname (usually something like 'sql123.infinityfree.com')
     - MySQL Username (e.g., 'if0_12345678')
     - MySQL Password (found in your account details)
     - MySQL Database Name (e.g., 'if0_12345678_carrental')
   - Go to "phpMyAdmin", select your new database, and Import the 'sqlfile/carrental.sql' file found in this folder.

3. Configuration
   - Edit the file 'includes/config.php' in this 'Car' folder.
   - Edit the file 'admin/includes/config.php' in this 'Car' folder.
   - Update the DB_HOST, DB_USER, DB_PASS, and DB_NAME with the values from Step 2.

4. Done!
   - Navigate to your URL to verify the site is live.
